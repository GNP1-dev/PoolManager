# PoolManager

A professional graphical desktop application for Cardano stake pool operators.

PoolManager provides a full GUI wrapper around Guild Operators cntools, replacing the terminal-based menu interface with a modern native desktop application. It connects securely to your Cardano node over SSH, meaning it runs on your everyday computer while all operations execute on your remote node.

## Built against
- cardano-node 10.5.3
- cardano-cli 10.11.1.0  
- cntools.library (Guild Operators) — 109 functions
- Tauri 1.6.6

## Attribution
All pool management logic is provided by Guild Operators cntools (Apache 2.0). PoolManager is a GUI wrapper only.
